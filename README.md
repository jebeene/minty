# minty

Minty is a simple personal finance tracking iOS app.

## Features

- Track income and expenses
- View statistical summaries 
- Data stored privately on your device  
- Built using Swift and SwiftUI  

## Demo
https://youtu.be/xIz01gv-e2A


## Installation

1. Clone the repository:
   ```
   git clone https://github.com/jebeene/minty.git
   cd minty
   ```

2. Open project in Xcode
   ```open minty.xcodeproj```
